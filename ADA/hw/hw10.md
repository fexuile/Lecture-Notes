## 10.1
$OPT(I) \ge 1, A(I) \le n-1, r = \frac{A(I)}{OPT(I)} \le n-1$.

对于一个特定的实例：菊花图，我们假设顶点1是最中心的点，剩余的点均和顶点1连边，$G=(V,E),|V|=n, |E| = n-1$。
那么有： 我们对于每一条边都不选择点1，$A(I) = n-1, OPT(I) = 1$。可以得到：
$r = \frac{A(I)}{OPT(I)} = n-1$

## 10.3
很明显，我们有：$OPT(I) \ge \frac{\sum w_i}{B}$。
接下来我们需要证明：$FF(I) < \frac{2\sum{w_i}}{B} \Leftrightarrow \frac{B}{2}\times FF(I) < \sum {w_i}$。
对于$FF(I)$方案中的分箱方案，任意两个箱子的重量和一定是大于$B$的，所以我们将他们两两匹配：
- $FF(I)$是奇数，我们将最大的抽取出来，设重量为$B' \ge \frac{B}{2}$，$\sum w_i > (FF(I) - 1)\times \frac{B}{2} + B' > FF(I)\times \frac{B}{2}$
- $FF(I)$是偶数，我们直接配对：$\sum w_i > \frac{B}{2}\times FF(I)$。

所以综上：
$\sum w_i > \frac{B}{2}\times FF(I) \Leftrightarrow FF(I) < \frac{2\sum w_i}{B}$。
$r = \frac{FF(I)}{OPT(I)} < 2 ， 也即FF(I) < 2OPT(I)$

## 10.4
证明：如果存在$r<\frac{3}{2}$近似比的多项式近似算法，设其为A，有：
我们考虑构造一个双机调度问题实例$I$:
1. $OPT(I) \le 2, r = \frac{A(I)}{OPT(I)} < \frac{3}{2} \rightarrow A(I) < 3$。
2. $OPT(I) \ge 3, r \ge 1 \rightarrow A(I) \ge OPT(I) \ge 3$。
也就是说通过该算法我们可以得到双机调度的判定，即：
如果$A(I) \ge 3$，则输出`no`，否则输出`yes`。
因为$A$算法是多项式时间的，所以双机调度是P问题。又因为由已知：双机调度为$NPC$的。
所以$P = NP$。
