### 3.3

我们设$f_i$表示长度为$i$的库房的最大收益，考虑一个类似于背包问题的$dp$过程：

$f_{j}=\max\{f[j-1], f_{j-l_i}+v_i\}$。

初始时$f$均为0

实际上就是一个01背包问题，伪代码如下：

```
for i in 1 -> n
	for j in D -> l_i:
		f[j] = max(f[j], f[j - l[i]] + v[i])
		f[j] = max(f[j], f[j-1])
```

时间复杂度最坏情况下为$O(nD)$

### 3.5

我们可以设$f_{j}$表示购买总价值为$j$的商品的最轻重量，那么：

$f_{j}=\min{\{f_j, f_{j-v_i} + w_i\}}$

初始值$f_{0}=0$，其他均为无穷大

这里相当于一个完全背包，伪代码如下：

```
for i in 1 -> n
	for j in v_i -> y:
		if f[j] > f[j - v[i]] + w[i]
			f[j] = f[j - v[i]] + w[i]
			cho[j] = i
```

最后我们根据$cho_j$就可以判断$j$重量选择的硬币是哪个，递归处理就能得到方案。

复杂度最坏为$O(ny)$，输出的复杂度也不会超过$O(ny)$。

### 3.7

#### （1）

我们设选取的建设点为$x_{k_1},x_{k_2},\dots,x_{k_m}$，其中$x_{k_1} < x_{k_2} < ...< x_{k_m}$，那么：

目标函数：$\max\sum_{i=1}^mp_{k_i}$

约束条件：$\forall i<m, x_{k_{i+1}}-x_{k_i} \ge d$

#### （2）

考虑动态规划，我们设$f_i$表示前$i$个位置开商店的最大收益，考虑对于当前$i$而言，有修建商店和不修建两种选择，那么：
$$
f_{i}=\max\{f_{i-1}, f_{j} + x_i(j < i, x_i-x_j \ge d)\}
$$
$f$初始化为0

最坏情况下时间复杂度为$O(n^2)$的。

### 3.12

我们考虑设$f_{l,r}$表示划分$l$~$r$这一段凸多边形的最小权值，答案存储在$f_{1,n}$中。

初始化$f$为正无穷，$f_{i,i}=0$，考虑转移：

$f_{l,r}=\min_{k=l+1}^{r-1}(f_{l,k}+f_{k,r}+d_{l,r}+d_{l,k}+d_{r,k})$。

要输出方案只需要记录每一个区间$[l,r]$对应的转移点$k$即可。

时间复杂度为$O(n^3)$。

### 3.15

设$f_{i,j}$表示当前在第$i$天，上一次检修在第$j$天的前$i$天加工数量最大值，$i \ge j$，有：


$$
f_{i,i} = \max_{j=1}^{i-1}{f_{i-1,j}}\\
f_{i,j} = f_{i-1,j}+\min\{s_{i-j}, x_i\}(j < i)
$$

初始时$f$为0

时间复杂度为:$O(n^2)$

### 3.17

设$f_{i,j}$表示到达当前点的路径数和最小值，则：
$$
f_{i,j} = \min{\{f_{i-1,j},f_{i,j-1}\}}+a_{i,j}
$$
初始值$f_{1,1}=a_{1,1}$，其他均为无穷大。

最后输出$\min_{i=1}^nf_{n,i}$即可。

时间复杂度为:$O(n^2)$
