# 宏观经济学概论
## 宏观经济学的概念
宏观经济：在这个经济单位内部，商品、人员、资本以及其他生产要素能够自由流动，而一旦超出这个经济单位的范围，这些要素的流动性就会受到限制。
北京和天津是同一个宏观经济，但是北京和纽约就不是。

中国由4个宏观经济组成：大陆、香港、澳门、台湾。

市场经济：三个市场
- 商品市场（流量，Q）：*新生产的产品*的交换市场。
- 资产市场（存量，K）：忽略各种资产之间的差别。
	- 存续时间。
	- 收益率，可能是负的。
	- 风险：高负利率出现的概率。
	- 流动性：卖来卖去的能力。
- 劳动力市场 (L)：忽略劳动力之间的差别。

资产：财富储存手段，财富实现跨时空的工具。
资产与商品的区别：在宏观经济学中资产是存量，商品是流量。商品在制造出来后是资产，成为存量的一部分。

例子：亚洲金融危机期间的外汇攻防
**卖空股票->卖出港元->港元贬值->加息->股市下跌->买回股票。**
这一系列依托于香港的`联系汇率制度`和`金融管理局制度`
> 香港的金融管理局是没有货币发行权的，想要进行防守只能够被迫干预市场，最终崩盘。

马来西亚的解决思路：关闭外汇市场，非常好的思路。

## 常用概念
- 存量 VS 流量：存量指某个实点上的状态，流量指变化量。
- 内生变量 VS 外生变量：内生变量指由模型内其他变量决定的，外生变量不由模型内其他变量决定。
- 均衡 VS 出清：均衡指的是一种平衡态，不一定保证供给量和需求量相等（存在短缺/过剩，但是价格稳定）。出清则是指完美均衡，在这时**总供给量恰好等于总需求量**。
> 出清又叫做“瓦尔拉斯均衡”，非出清的均衡叫做“非瓦尔拉斯均衡”。

# 宏观经济数据
## 产出与收入
经济在一定时期内的总产量，说明经济的总规模。
从数量指标变化到价值指标，用一个单一的数字指标来表达，具有简单性。
### 国内生产总值

**国内生产总值**：是一个国家在一定时期内(年、月或季度)生产出来的全部**最终**产品和服务的价值，在一定时间段内是流量。
社会总产值：将经济中所有经济单位的产值加总，在不同经济内不具有可比性。

计算方法：
1. 支出法：把对所有最终产品的支出加总。很难判定一个物品是中间产品还是最终产品。
2. 增加值：在生产的每一阶段上新创造出来的价值。
3. 收入法：将一个经济中所有企业的要素收入、折旧和间接税加总
> 直接税：纳税人就是负税人
> 间接税：纳税人不一定是负税人，大概率是消费者
#### 最终产品/服务

- 中间产品 / 服务：被作为一种投入品来生产其他产品和服务，并被一次性消耗掉的那些产品和服务
- 最终产品 / 服务：被直接用于消费或投资的产品和服务。
经济结构的变动会导致不同时期的数据也具有不可比性。

#### 当期产出
GDP是当期生产出来的产品的价值，因此在GDP中不计入就已有商品进行的交易。
比如说股票中的价值增值不算GDP，旧房交易不算GDP（但是中介的服务算GDP）

#### 市场价格
GDP是用各产品的市场价格来计算的,价格水平的变动会导致GDP数值的变动.

#### 存货的处理
增值部分计入今年的GDP。

#### GDP VS GNP
国内生产总值（GDP） VS 国民生产总值（GNP）

GDP = 本国生产要素在国内的收入＋外国生产要素在本国的收入＋折旧＋间接税
GNP = 本国生产要素在国内的收入＋本国生产要素在国外的收入＋折旧＋间接税

> 国内生产净值指从GDP中减去折旧（在生产过程中，总会出现资本的损耗。这种资本的损耗就是折旧）。

#### GDP的区别
名义GDP VS 真实GDP
- 剔除了价格变化影响之后的GDP数值叫做真实GDP
- 未剔除价格变化影响的GDP数值(即按当期价格计算的GDP数值)叫做名义GDP

GDP缩胀指数(GDP deflator)：名义GDP / 真实GDP * 100%
经济增长率：真实GDP的增长率。

#### GDP的缺点
1. 很粗略
2. GDP指标中没有反映产品的质量。
3. GDP指标不反映由技术进步所导致的产业结构变化对总经济福利的影响
4. GDP只计结果，不计生产过程中的损失，也不考虑某些生产的目的（污染和犯罪）
5. 用GDP来衡量经济福利，没有考虑到闲暇对福利的影响

### 支出的构成
对本国最终产品的支出可以分解为四个部分：
1. 居民户的消费支出
2. 企业的投资支出
3. 政府对商品和服务的购买
4. 国外对本国产品和服务的购买，即出口

> 题外话：房地产行业的下行和19年疫情其实关系不大，仍然是人口的关系。
#### 消费支出
本国居民户对最终产品和服务的支出。对本国产品和服务的购买构成对本国的需求，对外国的……构成对外国的需求
居民户对本国产品的支出为：$C_d=C-C_f$

#### 投资支出
投资指的是使得企业的生产能力提高的那些支出，和个人的投资（股票、债券）不同。
企业对本国投资品的购买为:$I_d=I-I_f$

#### 政府购买
政府购买分为：政府购买性支出和政府转移性支出
政府对本国产品和服务的支出$G_d=G-G_f$

#### 出口
出口用X表示，外国居民或团体对本国产品和服务的购买。

总支出：$AE=C_d+I_d+X+G_d$，但是这个公式很难使用。
转换一下：
$AE=(C-C_f)+(I-I_f)+(G-G_f)+X=C+I+G+X-(C_f+I_f+G_f)=C+I+G+(X-M)$
本国的总进口$M=C_f+I_f+G_f$，用NG表示净出口$NX=X-M$，所以最后：$AE=C+I+G+NX$
### GDP和个人可支配收入
GDP只有个人可支配收入部分影响消费。

1. 将GDP转换成GNP：去除外国要素的收入，加上本国要素在国外的要素
2. 剔除折旧
3. 政府通过税收 / 转移支付影响个人可支配收入
4. 企业需要剔除净收入，加上转移支付

所以最后得到：个人可支配收入＝GDP＋本国要素从国外得到的收入－外国要素从本国得到的收入－折旧－税收＋政府转移支付－企业保留利润＋企业转移支付
通常可以将其中一些抵消，得到：个人可支配收入=GDP＋政府转移支付－税收

使用上可以得到：个人可支配收入=消费＋储蓄

可以得到这些公式：
$$
\begin{aligned}
GDP&=C+I+G+NX \\
Y_d&=GDP+TR-TA\\
Y_d&=C+S\\
C+I+G+NX&=C+S+TA-TR
\end{aligned}
$$

## 价格指数
表示的是整个经济的平均价格水平。
上述说到的GDP Deflator实际上是一种价格指数，给出了某一期相对于基期的价格水平。
### 消费品价格指数（CPI）
给定一组消费品，各期成本与基期成本之比\*100就是消费品价格指数。
**注意：这一组消费品一定不能变**

### CPI和GDP膨胀指数
1. GDP缩胀指数中包括了一个经济生产出来的所有产品和服务，而CPI中仅包括消费品。所以如果投资品价格上涨GDP膨胀指数变大，但是CPI不变。
2. GDP缩胀指数只考虑一个经济本身生产出来的产品和服务，而不考虑进口品。所以进口消费品价格上涨，GDP膨胀指数不变，CPI变大。
3. CPI以基期的产品组合为基准，而GDP缩胀指数则以基期的价格水平为基准

### 通货膨胀率
某期通货膨胀率＝(本期价格指数－上期价格指数)/上期价格指数\*100%
针对不同的研究对象有不同的价格指数：
- 整个宏观经济：GDP Deflator
- 消费品：CPI
- 生产者的成本和购买的中间产品：PPI

## 资源利用情况
经济资源分为：劳动力、资本、土地和企业家才能
关心的问题在于：资源的利用率和利用效率

### 资源利用率
考虑资源利用率问题时，劳动力资源的利用率问题又占据核心地位。劳动力的失业问题一直是宏观经济学研究的主要任务之一。
#### 就业的确定
- 家庭(household)调查
- 机构(establishment)调查

可以将工作年龄人口（处于法定工作年龄段的人口数）分为：劳动力人口（就业者 / 失业者） / 非劳动力人口

非劳动力人口包括：学生、在家照看孩子的、生病的人、没有工作也没有工作意愿的人
**劳动力**（Labor Force）：工作年龄人口中愿意工作的人口总数，劳动力参与率＝劳动力/工作年龄人口。

#### 失业的确定
失业人口：愿意工作但是没有工作的人的数量。
**失业率** = 失业人数 / 劳动力 = 失业人数 / （失业人数 + 就业人数）

**自然失业率**：一个经济在任何时刻都会有失业存在
- 摩擦性失业：上述的那些人（应届生、重新进入劳动力市场的人、追求更好工作的人）从进入劳动力市场或离开原有工作到重新找到工作，需要一段时间，这就是摩擦性失业。
- 结构性失业：劳动力需求和供给的结构不相吻合。

当一个经济的劳动力市场达到均衡状态时，就说这个经济已经实现了充分就业
> 劳动力市场达到均衡：劳动力 = 提供岗位数，不需要考虑结构

自然失业率的测定：
- 理论性：非加速通货膨胀条件下的失业率。
- 实践性：如果选择足够长的一段时间，而且在这段时间内经济所经历的过热和衰退的次数相等，那么这段时间内的平均失业率接近自然失业率。
### 资源利用效率
资源利用效率是用生产率来衡量的
**劳动生产率** = 真实GDP / 总工时
**劳均GDP** = 真实GDP / 总劳动人口
**人均GDP** = 真实GDP / 总人口

**全要素生产率（TFP）**：将多种要素的使用构成一个单一的要素投入指数，要考虑到所有要素的生产率。
需要构造合理的全要素投入指数：科布-道格拉斯要素投入指数：$M=K^\alpha L^\beta$，K代表资本，L代表劳动力。
$TFP=Y/M$，即要素越多，生产率越低。

# 收入—支出模型
*假定价格不变在此意味着企业在给定价格下愿意提供任何数量的产品，因而总供给曲线是一条水平线。*

## 总支出与总需求
总支出：AE = C + I + G + NX
总需求：
1. 消费支出是消费者根据自己的效用最大化原则作出的决策，归于总需求
2. 政府购买也应归入总需求
3. 净出口也应全部归入总需求
4. 企业的固定资产投资应全部归入总需求，企业的自愿库存投资也应该归入总需求
	> 自愿投资＝实际投资支出－非自愿库存投资＝固定资产投资＋自愿存货投资

AD = C + $I_自$ + G + NX

## 消费函数与均衡
### 消费函数
假设价格水平不变，消费需求的主要决定因素是：收入水平，消费偏好。
$$C=C_0 + cY$$，C表示总消费支出，Y表示总收入，c叫做边际消费倾向（0~1），它表示人们收入的增加量中用于消费的部分所占的比例。
$C_0$是自发性消费需求，表示没有收入也需要的消费部分（或者说是与收入/消费偏好无关的部分）。
### 消费和存储
由：$Y=C+S$，可以得到：
$$
S=Y-C=-C_0+(1-c)Y
$$
所以1－c被叫做边际储蓄倾向，也是处于(0,1)之内。
### 总需求
假定政府购买和净出口均为0，投资函数是一个外生变量$I=I_0$，那么有：
$$
\begin{aligned}
AD&=C+I=C_0+I_0+cY \\
&=A_0+cY
\end{aligned}
$$
### 均衡收入与均衡产出
$AS=GDP=Y$
$AD=AS$，可以得到$Y=AD=A_0+cY$，解得
$Y^*=\frac{A_0}{1-c}$。
边际消费倾向越大，均衡收入越高；若给定边际消费倾向，则自发性支出越大，均衡收入越高
**越花钱，越有钱**
